I, Jurandir, developed this CRUD system with the goal of automating my weight loss process, facilitating the daily tracking of my weight. The project was structured in a modular and scalable way, using Python and Flask, which allows for efficient and organized management of my health data. To ensure the robustness and security of the application, I implemented SQLAlchemy 2.0 as an ORM, integrating a PostgreSQL database hosted on Neon.tech. The architecture follows professional standards, with isolation of environment variables and secure credential management, preventing failures and ensuring data integrity.. The deployment was carried out on Google Compute Engine using Gunicorn, ensuring stability in execution. More than just a simple log, this tool transforms numbers into strategic insight, allowing for trend analysis and maintaining the discipline necessary to achieve my goals. It is a solution that unites software development and self-care, providing precise and continuous control of my physical evolution.
